## name
floor
## title
The floor function.
## description
Rounds a number down to the nearest integer.
## details
The floor function takes as input a real number x, and gives as output the
greatest integer less than or equal to x.
## authors
## see_also
trunc
## example
    x <- 2.4
    floor(x) # returns 2
    
    y <- -2.4
    floor(y) # returns -3
