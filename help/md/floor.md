## name
floor
## title
The floor function.
## description
Rounds a number down to the nearest integer.
## details
The floor function takes as input a real number x, and gives as output the
greatest integer less than or equal to x. For example: ⌊2.4⌋ = 2, ⌊−2.4⌋ = −3.
