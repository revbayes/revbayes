## name
matrix
## title
Matrix
## description
Create matrix object.
## details
This function creates a matrix object from a 2-dimensional set of real numbers.
## authors
Michael Landis
## see_also
MatrixReal
## example
    # create set of real numbers to convert to matrix
    x <- [[1, 1], [1, 1]]
    # convert x to matrix object
    y <- matrix(x)
