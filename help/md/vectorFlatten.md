## name
vectorFlatten
## title
Vector Flatten
## description
Flatten a vector to one dimension.
## details
This function accepts a two-dimensional vector as an argument and flattens it
to one dimension.
## authors
Michael Landis
## see_also
v
RealPos
## example
    # Define Vector
    x <- v([1, 2], [3, 4])
    # Flatten
    x_flat <- vectorFlatten(x)
    x_flat
    [1, 2, 3, 4]
